
файлы прошивки для обновления на новые версии (Вторая версия прошивки не подошла)
Название платы - mks robin nano V1.2
https://github.com/makerbase-mks/MKS-Robin-Nano-V1.X/tree/master/Nano%20firmware%20v2.x.x


Для прошивки - 

1. скопировать файлы из папки v2.x Marlin official на карту памяти
2. Выключить принтер
3. Вставить карту памяти и включить принтер
- произойдёт загрузка прошивки
4. Вытащить карту памяти



Для изменения настроек принтера требуется отредактировать файл 

1. Скопировать файл robin_nano35_cfg в любое место (файл находится внутри папки v2.x Marlin official)
2. Изменить файл, открыв его при помощи блокнота или другого текстового редактора
3. Скопировать файл на SD карту
4. Вставить карту в принтер и включить его
-- Если всё пройдёт удачно - принтер переименует файл
